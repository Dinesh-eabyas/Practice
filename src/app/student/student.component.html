This is First Component

We can represent selectors in ".ts" file for any Component in 3 Ways.

Selector: 'app-student';
In HTML we have to write as,

"'app-student' '/app-student'"

Selector: '.app-student';
In HTML we have to write as,

"'div class="app-student'" '/div'"

Selector: '[app-student]';
In HTML we have to write as,

"'div app-student' '/div'"
